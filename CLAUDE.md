# CLAUDE.md

You are an expert LLM application developer. This is Criss36's LLM career portfolio project.

## Project Structure

- `dist/` - Production build of the portfolio website
- `docs/` - Technical deep-dive documents
- `src/` - Source code (see llm-career-portfolio)

## How to Run

```bash
cd llm-career-portfolio
npm install
npm run dev     # Development server
npm run build   # Production build
```

## Tech Stack

- Frontend: React 18 + TypeScript + Vite + TailwindCSS
- Backend concepts: LangChain, CrewAI, RAG, Multi-Agent

## Key Principles

1. Production-first: Every demo must be runnable and deployed
2. RAG quality: Recall rate > 90% in production
3. Agent reliability: Structured output, error handling, observability
4. Documentation: Every project has a deep-dive technical blog post
